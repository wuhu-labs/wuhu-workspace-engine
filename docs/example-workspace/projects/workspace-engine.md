# Workspace Engine

The core query engine for Wuhu's workspace system.

## Goals

- In-memory SQLite for fast queries
- Live filesystem watching
- Cross-platform (macOS + Linux)

## Status

Feature-complete for v0.1.0. Next milestone focuses on performance optimization
and adding computed/virtual columns.
